API Server Activation Guide

Streamdek works with pear-desktop's API Server. Choose one of these methods:

Option A — No Authorization (recommended)

  1. Open pear-desktop
  2. Go to Settings → API Server
  3. Check Enable API Server
  4. Set Authorization to "No authorization"
  5. Set port to 26538 (default)
  6. In Streamdek, leave "Use authentication" unchecked
  7. Click Probe Connection — Streamdek connects instantly

Option B — AUTH_AT_FIRST

  1. Set pear API Server Authorization to "AUTH_AT_FIRST"
  2. In Streamdek, check "Use authentication"
  3. Click Connect & Authorize
  4. pear-desktop will show a dialog — click Allow
  5. Streamdek connects automatically

If auto-discovery fails, make sure pear-desktop is running and the API server is enabled. The plugin probes localhost:26538 automatically.
