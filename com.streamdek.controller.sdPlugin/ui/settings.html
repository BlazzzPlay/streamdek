API Server Activation Guide

Streamdek Controller connects to pear-desktop's API Server with no authorization.

Setup Instructions

  1. Open pear-desktop
  2. Go to Settings → API Server
  3. Check Enable API Server
  4. Set Authorization to "No authorization"
  5. Set port to 26538 (default)
  6. In Streamdek Controller, click Probe & Connect
  7. Streamdek connects instantly — ready to use

If auto-discovery fails, make sure pear-desktop is running and the API server is enabled. The plugin probes localhost:26538 automatically.
